Introduction

Sequelize is a powerful Javascript library built on Node.js, which makes the developer's life easier by helping to manage SQL databases.

Sequelize.js is an ORM (Object/Relational Mapper) which provides easy access to MySQL, MariaDB, SQLite or PostgreSQL databases by mapping database entries to objects and vice versa. It has a very powerful migrations mechanism that can transform an existing database schema into a new version. It also provides database synchronization mechanisms that can create a database structure by specifying the model structure.
